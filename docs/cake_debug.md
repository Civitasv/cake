# cake-run

## NAME

cake-debug -- Debug the current package

## SYNOPSIS

`cake debug [options]`

## DESCRIPTION

Debug a binary of the local package.

## OPTIONS

### Target Selection

`--bin` *name*: Run the specified binary.

`--debugger` *name*: Specify debugger, support "gdb", "lldb" or "code"

`--args` args: Arguments passed to binary.
