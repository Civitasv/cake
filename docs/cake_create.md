# cake-create

## NAME

cake-create -- Create a template.

## SYNOPSIS

`cake create [options]`

## DESCRIPTION

Create a template.

## OPTIONS

### Template Selection

`--template <type>` : Create specific type of template using cake.

### Common Options

`--config` *KEY=VALUE*

All options will send to cake, using `-DKEY=VALUE`.

`--help`: Prints help information.
